Court Booking (Badminton & Pickleball) - Java 21 (Spring Boot)

This is a minimal Spring Boot + static frontend example demonstrating a court booking system. Run with Java 21 and Maven.

Run locally

Install Java 21 and Maven.
Build: mvn clean package
Run: mvn spring-boot:run or java --enable-preview -jar target/court-booking-0.1.0.jar
Open: http://localhost:8080/
The frontend is a simple static index.html in src/main/resources/static/.
